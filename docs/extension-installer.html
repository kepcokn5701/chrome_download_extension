버전 0.2.2 — 사내망 설치 페이지

1설치 폴더 선택 후 파일 풀기

아래 버튼을 누르면 폴더 선택창이 뜹니다. 새로 만든 빈 폴더를 선택하세요 (예: Documents\srm-extension).

이 브라우저는 폴더 직접 쓰기를 지원하지 않습니다. 아래 링크를 모두 클릭해 같은 폴더(예: Documents\srm-extension)에 다운로드 받으세요. icons/로 시작하는 파일은 그 폴더 안에 icons 하위 폴더 만들어 넣어주세요.

💡 폴더는 옮기지 않을 안정적 위치에 만드세요 (예: D 드라이브 또는 Documents). 폴더를 옮기거나 지우면 확장이 동작하지 않습니다.

2확장 프로그램으로 로드

  1. Edge 주소창에 edge://extensions/ 입력 (Chrome은 chrome://extensions/)
  2. 우측 상단 "개발자 모드" 토글 ON
  3. "압축해제된 확장 프로그램을 로드합니다" 클릭
  4. 1단계에서 선택한 폴더 선택 → 끝
