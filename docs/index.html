1

다운로드 후 압축 해제

위 ⬇ 다운로드 (.zip) 버튼으로 받은 zip 파일을 원하는 위치에 압축 해제하세요.

예: D:\chrome_download_extension 또는 C:\Users\본인\Documents\chrome_download_extension

⚠ 주의: 폴더를 다른 곳으로 옮기거나 삭제하면 확장이 동작 안 합니다. 옮기지 않을 위치에 두세요.

chrome://extensions/

자주 묻는 질문

회사 보안에 걸리지 않나요?

걸리지 않습니다. 이 도구는 자동화 도구가 아니라 본인 Chrome 안에서 본인이 클릭한 결과로 동작합니다. Power Automate Desktop이나 DownThemAll!과 같은 원리입니다. SRM의 봇 차단 정책에 걸리지 않습니다.

Chrome이 "여러 파일 다운로드 허용?" 안내를 띄웁니다.

일괄 다운로드 시 첫 실행에서 한 번 뜹니다. "허용" 클릭하면 그 사이트(SRM)에서는 다음부터 안 뜹니다.

다운로드 위치를 D 드라이브로 바꾸고 싶어요.

이 확장 자체는 Chrome의 기본 Downloads 폴더 하위로만 저장 가능합니다 (Chrome 보안 제약). 위치 자체를 바꾸려면:

  1. Chrome 주소창에 chrome://settings/downloads 입력
  2. "위치" 옆 변경 클릭
  3. D:\srm_downloads 같은 폴더 선택

이렇게 하면 우리 도구의 다운로드도 그 위치로 자동으로 갑니다.

다운로드한 PDF/HWP가 다음 날 사라졌어요.

사내 보안정책상 Desktop 등 일부 위치의 PDF/HWP/ZIP은 매일 자동 삭제됩니다. 같은 날 안에 검토 또는 D 드라이브/OneDrive로 백업하세요.

일부 파일만 받아졌어요.

큰 파일(수십 MB)은 SRM이 처리에 시간이 걸립니다. 이 도구는 파일 크기에 따라 자동으로 클릭 간격을 조절(250ms~3.5초)하지만, 사이트가 매우 느릴 땐 누락 가능. popup의 로그에서 누락 파일 확인 후 같은 화면에서 다시 누르면 받아집니다 (덮어쓰기).

"확장 프로그램 로드 실패" 에러가 떠요.

다음을 확인하세요:

  • 선택한 폴더 안에 manifest.json 파일이 있는지
  • zip을 압축 해제할 때 폴더가 한 단계 더 안에 있을 수 있음 (예: chrome_download_extension-main/chrome_download_extension-main/) — 가장 안쪽 폴더 선택
업데이트가 있으면 어떻게 받나요?

이 페이지에서 다시 zip 다운로드 → 기존 폴더 덮어쓰기 → chrome://extensions/에서 이 확장의 새로고침 아이콘(🔄) 클릭. 끝.
